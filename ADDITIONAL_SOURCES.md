# Дополнительные источники спортивных событий

## Текущие источники

### ✅ KudaGo API (реализовано)
- **URL**: https://kudago.com/public-api/v1.4/events/
- **Категории**: concert, sport, exhibition, festival
- **Охват**: Концерты, спортивные события, выставки, фестивали
- **Обновление**: Каждые 6 часов
- **Преимущества**: Официальный API, стабильный, хорошая документация
- **Недостатки**: Не все спортивные события попадают в базу

## Планируемые источники

### 1. Sports.ru
- **URL**: https://www.sports.ru/calendar/
- **Тип**: Спортивный календарь
- **Охват**: Футбол, хоккей, баскетбол, все виды спорта
- **Реализация**: Парсинг HTML или поиск API
- **Приоритет**: Высокий

### 2. Championat.ru
- **URL**: https://www.championat.com/calendar/
- **Тип**: Спортивный календарь
- **Охват**: Все виды спорта, особенно футбол и хоккей
- **Реализация**: Парсинг HTML
- **Приоритет**: Высокий

### 3. Kassir.ru (Билетная касса)
- **URL**: https://www.kassir.ru/
- **Тип**: Платформа продажи билетов
- **Охват**: Спорт, концерты, театр
- **Реализация**: Парсинг или API (если доступен)
- **Преимущества**: Точные даты и места проведения
- **Приоритет**: Средний

### 4. Ticketland.ru
- **URL**: https://ticketland.ru/
- **Тип**: Билетная платформа
- **Охват**: Спортивные события, концерты
- **Реализация**: Парсинг HTML
- **Приоритет**: Средний

### 5. Официальные сайты команд

#### Футбол (РФПЛ)
- **Спартак**: https://spartak.com/calendar/
- **ЦСКА**: https://pfc-cska.com/calendar/
- **Динамо**: https://fcdynamo.ru/calendar/
- **Локомотив**: https://fclm.ru/calendar/

#### Хоккей (КХЛ)
- **Спартак**: https://hcspartak.ru/calendar/
- **ЦСКА**: https://hccska.ru/calendar/
- **Динамо**: https://hcdynamo.ru/calendar/

**Реализация**: Парсинг календарей команд
**Приоритет**: Средний
**Преимущества**: Самая точная информация о домашних матчах

### 6. VK Events API
- **URL**: https://dev.vk.com/method/groups.getById
- **Тип**: События сообществ ВКонтакте
- **Охват**: Официальные страницы команд и стадионов
- **Реализация**: VK API
- **Приоритет**: Низкий
**Требования**: VK API токен

### 7. Timepad.ru
- **URL**: https://timepad.ru/
- **Тип**: Платформа событий
- **Охват**: Различные мероприятия, включая спорт
- **Реализация**: API или парсинг
- **Приоритет**: Низкий

### 8. Яндекс.Афиша (уже в коде, но deprecated)
- **URL**: https://afisha.yandex.ru/moscow/sport
- **Тип**: Афиша событий
- **Охват**: Спорт, концерты, театр
- **Реализация**: Парсинг HTML (сложный)
- **Статус**: Можно реактивировать для спорта
- **Приоритет**: Низкий (сложный парсинг)

### Дедупликация событий

Важно избегать дубликатов при парсинге из нескольких источников:
- Сравнение по названию события
- Сравнение по времени (±1 час)
- Сравнение по стадиону/зоне

### Частота обновления

- **KudaGo**: каждые 6 часов (текущая настройка)
- **Sports.ru/Championat**: каждые 12 часов (меньше изменений)
- **Официальные сайты команд**: каждые 24 часа (календарь стабилен)
- **Билетные платформы**: каждые 12 часов

## Юридические аспекты

⚠️ **Важно**: При парсинге сайтов необходимо:
1. Проверить robots.txt
2. Соблюдать rate limits
3. Использовать User-Agent
4. Не перегружать серверы запросами
5. Предпочитать официальные API, где доступны

## Мониторинг качества данных

После добавления новых источников отслеживать:
- Количество событий из каждого источника
- Процент дубликатов
- Точность определения стадиона/зоны
- Количество событий без зоны (пропущенных)
