## 🔍 Возможные проблемы

### Если форма не открывается
**Причина:** Элемент `#newEventModal` не найден

**Проверка:**
```javascript
// В консоли браузера (F12):
document.getElementById('newEventModal')
// Должно вернуть элемент, а не null
```

**Решение:** Проверить что HTML содержит:
```html
<div id="newEventModal" class="modal">
```

### Если список не загружается
**Причина:** API endpoint не отвечает или возвращает ошибку

**Проверка:**
```javascript
// В консоли браузера:
fetch('/admin/api/events', {
  headers: { 'Authorization': 'Bearer ' + localStorage.getItem('admin_token') }
}).then(r => r.json()).then(console.log)
```

**Ожидается:** Массив мероприятий `[{id: 1, name: "...", ...}]`

### Если данные отображаются как `${variable}`
**Причина:** Старая версия кода в браузере (кэш)

**Решение:**
1. Очистить кэш браузера (Ctrl+Shift+Delete)
2. Жесткое обновление (Ctrl+Shift+R)
3. Или открыть в режиме инкогнито (Ctrl+Shift+N)
