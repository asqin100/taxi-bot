## Почему проблема может сохраняться

1. **Бот не обновлен на сервере**
   - Последний коммит: 556fa03
   - Нужно выполнить: `git pull && systemctl restart taxibot`

2. **Кеш Telegram клиента**
   - Telegram может кешировать layout клавиатур
   - Решение: полностью закрыть и открыть бот заново

3. **Клавиатура выбора времени**
   - `quiet_hours_keyboard()` имеет 6 кнопок в ряд (для выбора часов)
   - Это может влиять на отображение других клавиатур в Telegram
   - Это необходимый дизайн, его нельзя изменить

## Команда для обновления на сервере

```bash
cd /opt/taxibot && git pull origin main && systemctl restart taxibot
```

## Тестирование

После обновления:

1. Полностью закрой бот в Telegram
2. Открой бот заново
3. Введи `/start` - все кнопки должны быть широкими
4. Зайди в "Мой кабинет" → "Финансы"
5. Вернись в главное меню - кнопки должны остаться широкими
6. Зайди в "Уведомления" → "Настроить время" (здесь будут узкие кнопки - это нормально)
7. Вернись в главное меню - кнопки должны быть широкими

## Если проблема сохраняется

Это может быть особенность Telegram клиента. Когда в одном чате есть клавиатуры с разной шириной кнопок (например, выбор времени с 6 кнопками в ряд), Telegram может применять этот layout к последующим клавиатурам.

Решение: не использовать клавиатуру выбора времени, или принять это как особенность Telegram.
