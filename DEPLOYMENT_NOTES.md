## Новые функции (Elite тариф - 999₽/мес)

### 1. CSV экспорт смен
- Команда: `/export` или кнопка в меню
- Экспорт за последние 30 дней
- Rate limiting: 1 раз/день для обычных пользователей, безлимит для Elite

### 2. Калькулятор налогов НПД
- Команда: `/tax`
- Расчет для самозанятых (4% физлица, 6% юрлица)
- Учет налогового вычета до 10,000₽

### 3. Тепловая карта заработка
- Команда: `/heatmap`
- Визуализация по часам × дням недели
- Анализ последних 90 дней

### 4. Расширенная статистика
- Периоды: 7/14/30 дней
- Детальная аналитика смен, заработка, километража

### 5. REST API
- Запуск: `python -m uvicorn bot.api.app:app --host 0.0.0.0 --port 8000`
- Документация: http://localhost:8000/api/docs
- Endpoints:
  - GET /api/v1/shifts/export - экспорт смен
  - GET /api/v1/stats/summary - статистика
  - GET /api/v1/predictions/demand - предсказания спроса

### 6. ML предсказания
- Модель: RandomForest
- Требует: 60 дней исторических данных
- Автоматический сбор данных каждые 8 минут
