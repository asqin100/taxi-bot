### 4. Тест через веб-интерфейс

#### 4.1 Откройте логи в отдельном терминале:

```bash
tail -f /opt/taxibot/bot.log | grep -i "event"
```

Оставьте этот терминал открытым!

#### 4.2 В браузере:

1. Откройте: https://kefpulse.ru/admin/dashboard
2. Войдите с паролем
3. Вкладка "🎭 Мероприятия"
4. Кнопка "➕ Добавить мероприятие"
5. Заполните:
   - **Название:** Тест
   - **ID зоны:** luzhniki
   - **Тип:** concert
   - **Время окончания:** ЗАВТРА 20:00 ⚠️ (не сегодня!)
6. Нажмите "✅ Создать мероприятие"

#### 4.3 Смотрите в терминал с логами

**Должны появиться строки:**
```
Received event creation request: {'name': 'Тест', ...}
Parsed end_time: 2026-03-13 20:00:00, current time: 2026-03-12 ...
Admin created event: Тест at luzhniki, ends at 2026-03-13 20:00:00
```

**Если ошибка:**
- Скопируйте ВСЕ строки из логов
- Отправьте мне

### 5. Тестирование задач из 3434.txt

#### ✅ Task #1: Сортировка в уведомлениях

- Дождитесь уведомления о высоких коэффициентах
- Нажмите "🔤 Сортировать по имени"
- Проверьте, что зоны отсортированы по алфавиту
- Нажмите "📊 Сортировать по коэффициенту"
- Проверьте, что зоны отсортированы по убыванию

#### ✅ Task #2: Поиск ресторанов

- Главное меню → "🍔 Заехать на обед"
- Отправьте геолокацию
- Должен показать рестораны в радиусе 10 км

#### ✅ Task #3: Бан-система

- Админ-панель → вкладка "🚫 Забаненные"
- Попробуйте забанить тестового пользователя
- Проверьте список забаненных

#### ✅ Task #4: Лимит "Куда ехать"

- Главное меню → "🔍 Куда ехать"
- Используйте 3 раза (для FREE тарифа)
- На 4-й раз должно показать "Лимит исчерпан"
- Проверьте счётчик "Использовано: X/Y"

#### ✅ Task #5: Счётчик в геоалертах

- Включите геоалерты
- Дождитесь уведомления
- Проверьте строку "📊 Использовано сегодня: X/Y"

#### ✅ Task #6: Кнопка VPN

- Откройте главное меню
- Проверьте, что кнопки "🔐 VPN" нет
