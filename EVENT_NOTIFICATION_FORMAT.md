# Формат уведомлений о мероприятиях

## Пример 1: Уведомление за 20 минут до окончания

```
⏰ Скоро закончится мероприятие!

🎵 Концерт Би-2
📍 Зона: Красногорск
⏰ Окончание: 22:30

💡 Ожидается высокий спрос на такси!
```

**Кнопки:**
- 🚗 Поехали (Навигатор) → открывает Яндекс.Навигатор с маршрутом
- 🗺 Поехали (Карты) → открывает Яндекс.Карты с маршрутом
- 🏠 Главное меню → возврат в главное меню бота

---

## Логика работы

1. **За 20 минут до окончания** - отправляется предупреждение
2. **При окончании** - отправляется уведомление с текущим коэффициентом
3. **Кнопки навигации** - ведут в зону мероприятия (координаты зоны)
4. **Фильтрация** - пользователь получает только те типы, которые выбрал в настройках

---

## Источник данных

Мероприятия загружаются из **KudaGo API**:
- Категории: concert, theater, festival, party, exhibition, show, kids, sport, entertainment
- Период: ближайшие 30 дней
- Лимит: 200 событий
- Обновление: автоматически через cron

---

## Настройки пользователя

В разделе "Уведомления" пользователь может:
- Включить/выключить уведомления о мероприятиях
- Выбрать типы мероприятий (концерты, спорт, театр и т.д.)
- Настроить тихие часы

---

## Реализация

Файлы:
- `bot/services/event_notifier.py` - отправка уведомлений
- `bot/services/event_parser.py` - парсинг мероприятий из KudaGo
- `bot/services/events.py` - работа с БД мероприятий
- `bot/models/event.py` - модель мероприятия
