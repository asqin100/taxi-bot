# Финальное тестирование бота - Чек-лист

## Задача 19/19: Проверить что все функции не конфликтуют

### Основные сценарии для тестирования

## 1. Первый запуск и онбординг

- [ ] `/start` - показывает онбординг для новых пользователей
- [ ] Кнопки главного меню все широкие (одинаковой ширины)
- [ ] Все кнопки кликабельны и не перекрываются

## 2. Навигация по меню

- [ ] "🗺 Куда ехать?" - запрашивает геолокацию
- [ ] После отправки геолокации - находит зоны с коэффициентом ≥1.3
- [ ] Показывает кнопки навигации (Яндекс.Навигатор / Яндекс.Карты)
- [ ] "⚡ Все функции" - показывает список функций по тарифу
- [ ] "🏆 ТОП-5 зон" - показывает топ зон с коэффициентами
- [ ] "🤖 AI-советник" - доступен для Pro+
- [ ] "🚦 Пробки" - показывает "🔒 Pro" для Free тарифа
- [ ] "📍 Геоалерты" - настройка геоалертов
- [ ] "🔔 Уведомления" - настройка уведомлений
- [ ] "👤 Мой кабинет" - открывает подменю (Финансы, Достижения, Челлендж)
- [ ] "⭐ Подписка" - показывает информацию о подписке
- [ ] "🎁 Промокод" - ввод промокода
- [ ] "💰 Реферальная программа" - показывает реферальную ссылку
- [ ] "⚙️ Настройки" - настройки бота
- [ ] "❓ Справка" - справочная информация

## 3. Мой кабинет

- [ ] Зайти в "👤 Мой кабинет"
- [ ] "💰 Финансы" - открывает финансовый трекер
- [ ] "🏅 Достижения" - показывает достижения
- [ ] "🏆 Челлендж" - показывает челлендж
- [ ] Кнопка "◀️ Назад" возвращает в главное меню
- [ ] После возврата кнопки остаются широкими (не становятся узкими)

## 4. Финансы

- [ ] "▶️ Начать смену" - начинает смену
- [ ] "⏹ Завершить смену" - запрашивает заработок
- [ ] "📊 Статистика" - показывает статистику
- [ ] "📥 Экспорт в CSV" - для Elite тарифа экспортирует данные
- [ ] "📥 Экспорт в CSV" - для Free показывает "🔒 Elite"
- [ ] После экспорта CSV есть кнопка "🏠 Главное меню"
- [ ] "💸 Расходы" - показывает настройки расходов
- [ ] "🎯 Цели" - показывает финансовые цели
- [ ] "🚗 Мой тариф" - выбор тарифа водителя

## 5. Пробки

- [ ] Free тариф: кнопка "🚦 Пробки 🔒 Pro" показывает сообщение о необходимости Pro
- [ ] Pro+ тариф: кнопка "🚦 Пробки" открывает меню пробок
- [ ] "🚦 Общая обстановка" - показывает пробки в Москве
- [ ] Время отображается в московском часовом поясе (не на 3 часа назад)
- [ ] "🛣 МКАД" - показывает пробки на МКАД
- [ ] "🔄 ТТК" - показывает пробки на ТТК
- [ ] Есть кнопка "◀️ Назад" и "🏠 Главное меню"
- [ ] При повторном запросе данные обновляются (кеш очищается)

## 6. Мероприятия

- [ ] `/events` - показывает список мероприятий
- [ ] Мероприятия из разных категорий (концерты, спорт, театр, выставки, шоу, детские, развлечения)
- [ ] Есть кнопка "🏠 Главное меню"

## 7. Геоалерты

- [ ] "📍 Геоалерты" - открывает настройки
- [ ] Free тариф: показывает "🔒 Pro"
- [ ] Pro+ тариф: можно включить/выключить
- [ ] "🔔 Включить геоалерты" - запрашивает Live Location
- [ ] После отправки Live Location - подтверждение получено
- [ ] "📍 Обновить Live Location" - запрашивает новую геолокацию
- [ ] "⚙️ Изменить порог коэффициента" - настройка порога

## 8. Уведомления о коэффициентах

- [ ] Настроить уведомления в "🔔 Уведомления"
- [ ] Выбрать зоны и тарифы
- [ ] Установить порог коэффициента
- [ ] Дождаться уведомления (если есть высокие коэффициенты)
- [ ] В уведомлении есть кнопка "🏠 Главное меню"
- [ ] Проверить задержки по тарифам:
  - [ ] Elite: мгновенно (0 сек)
  - [ ] Premium: 60 сек задержка
  - [ ] Pro: 90 сек задержка
  - [ ] Free: 120 сек задержка

## 9. Подписки и оплата

- [ ] "⭐ Подписка" - показывает текущую подписку
- [ ] "📊 Сравнить тарифы" - показывает сравнение
- [ ] "⬆️ Улучшить тариф" - показывает варианты оплаты
- [ ] Выбрать тариф (Pro/Premium/Elite)
- [ ] Кнопка "💰 Оплатить балансом" (если есть баланс)
- [ ] Кнопка "💳 Оплатить" (не "Оплатить картой")
- [ ] НЕТ кнопки "тест платеж 5₽"
- [ ] После успешной оплаты есть кнопка "📊 Главное меню"

## 10. Robokassa чеки

- [ ] При создании платежа формируется Receipt
- [ ] Receipt содержит:
  - [ ] sno: "usn_income"
  - [ ] items с name, quantity, sum
  - [ ] payment_method: "full_prepayment"
  - [ ] payment_object: "service"
  - [ ] tax: "none"
- [ ] Receipt включается в подпись
- [ ] Receipt URL-кодируется

## 11. Реферальная программа

- [ ] "💰 Реферальная программа" - показывает реферальную ссылку
- [ ] Показывает баланс
- [ ] Показывает статистику рефералов
- [ ] "💰 Оплатить балансом" - оплата подписки балансом

## 12. Промокод

- [ ] "🎁 Промокод" в главном меню
- [ ] Ввод промокода работает
- [ ] Активация промокода применяет бонусы

## 13. Карта

- [ ] "🗺 Открыть карту" - открывает WebApp
- [ ] Карта загружается по адресу https://kefpulse.ru
- [ ] На карте отображаются гексагоны с коэффициентами
- [ ] Коэффициенты обновляются (не пустая карта)

## 14. Конфликты и баги

- [ ] Кнопки не меняют размер при навигации по меню
- [ ] Нет дублирующихся кнопок
- [ ] Все callback_data уникальны
- [ ] Нет ошибок в логах при навигации
- [ ] Бот не "молчит" после отправки геолокации
- [ ] Время в пробках корректное (московское, не -3 часа)
- [ ] Уведомления приходят с правильными задержками
- [ ] Free тариф не имеет доступа к Pro функциям

## 15. Проверка логов

```bash
# Проверить логи на ошибки
journalctl -u taxibot -n 200 | grep -i "error\|exception\|traceback"

# Проверить работу коэффициентов
journalctl -u taxibot -n 100 | grep -i "surge\|coefficient"

# Проверить работу уведомлений
journalctl -u taxibot -n 100 | grep -i "notification\|alert"
```

## 16. Производительность

- [ ] Бот отвечает быстро (< 2 сек на команды)
- [ ] Карта загружается быстро
- [ ] Нет зависаний при навигации
- [ ] Память и CPU в норме

- [ ] Бот не запускается
- [ ] Ошибки при оплате
- [ ] Не работают уведомления
- [ ] Карта не открывается
- [ ] Коэффициенты не загружаются

- [ ] Мелкие UI недочеты
