# Разделение функций геолокации

## Две разные функции

### 1. "Куда ехать?" (Where to go)

**Назначение:** Разовый поиск зоны с высоким коэффициентом

**Как работает:**
- Пользователь нажимает "🗺 Куда ехать?" в главном меню
- Бот просит поделиться геопозицией (одноразово)
- Бот ищет зоны с коэффициентом ≥ 1.3 в радиусе **10 км**
- Показывает ближайшую зону с кнопками навигации
- Геопозиция **НЕ сохраняется** в базу данных

**Параметры:**
- Радиус поиска: **10 км**
- Минимальный коэффициент: **1.3x**
- Тип геолокации: **одноразовая** (не live)
- Доступ: **все пользователи**

**Пример сообщения:**
```
✅ Найдена зона с высоким коэффициентом!

📍 Зона: Красногорск
💰 Коэффициент: 2.30x
📏 Расстояние: 8.5 км

Выберите приложение для навигации:
[🚗 Яндекс Навигатор] [🗺 Яндекс Карты]
```

Если не найдено:
```
😔 Высокий коэффициент не найден

К сожалению, в радиусе 10 км от вас нет зон
с коэффициентом ≥ 1.3.

💡 Попробуйте позже или переместитесь в другой район
```

---

### 2. "Геоалерты" (Geo alerts)

**Назначение:** Автоматические уведомления о высоких коэффициентах рядом

**Как работает:**
- Пользователь включает Live Location на 8 часов
- Бот сохраняет геопозицию в базу данных
- Каждые 2 минуты проверяет зоны в радиусе **5 км**
- Если находит зону с коэффициентом выше порога → отправляет алерт
- Cooldown: не более 1 алерта на зону за 15 минут

**Параметры:**
- Радиус поиска: **5 км**
- Минимальный коэффициент: **настраивается пользователем** (1.2x - 2.5x)
- Тип геолокации: **live** (8 часов)
- Проверка: **каждые 2 минуты**
- Доступ: **Pro, Premium, Elite**

**Пример алерта:**
```
🔥 ВЫСОКИЙ КОЭФФИЦИЕНТ РЯДОМ!

📍 Зона: Красногорск
💰 Коэффициент: x2.30
🚗 Тариф: Эконом
📏 Расстояние: 3.5 км

Выберите приложение для навигации:
[🗺 Яндекс.Карты] [🧭 Яндекс.Навигатор]
[🔕 Отключить геоалерты] [◀️ Главное меню]
```

## Пользовательские сценарии

### Сценарий 1: Быстрый поиск зоны
1. Водитель хочет узнать, куда ехать прямо сейчас
2. Нажимает "🗺 Куда ехать?"
3. Делится геопозицией (одноразово)
4. Получает результат в радиусе 10 км
5. Едет в указанную зону

### Сценарий 2: Работа со включенными геоалертами
1. Водитель включает Live Location на 8 часов
2. Начинает работать, перемещается по городу
3. Каждые 2 минуты бот проверяет зоны в радиусе 5 км
4. Когда рядом появляется высокий коэффициент → приходит алерт
5. Водитель едет в зону по навигации

---

## Преимущества разделения

1. **"Куда ехать?"** - для тех, кто не хочет делиться live location
2. **Больший радиус** (10 км) для разового поиска
3. **Меньший радиус** (5 км) для геоалертов = более релевантные уведомления
4. **Нет спама** - геоалерты только для подписчиков
5. **Приватность** - одноразовая геопозиция не сохраняется
