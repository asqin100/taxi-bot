# gstack Skills - Windows Compatibility

## ✅ РАБОТАЮТ НА WINDOWS (8 skills)

### 1. /review
**Что делает:** Параноидальное код-ревью
**Использование:** `/review`
**Проверяет:**
- Race conditions
- N+1 queries
- SQL injection
- Security issues
- Test coverage

### 2. /ship
**Что делает:** Автоматизация релиза
**Использование:** `/ship`
**Действия:**
- Синхронизирует с main
- Запускает тесты
- Пушит ветку
- Создает PR

### 3. /plan-eng-review
**Что делает:** Технический архитектор
**Использование:** `/plan-eng-review`
**Создает:**
- Архитектурные диаграммы
- Граничные случаи
- Failure modes
- Тестовую стратегию

### 4. /plan-ceo-review
**Что делает:** Продуктовое мышление
**Использование:** `/plan-ceo-review`
**Проверяет:**
- Правильность продуктового видения
- Поиск 10-star версии
- User empathy

### 5. /retro
**Что делает:** Еженедельная ретроспектива
**Использование:** `/retro`
**Анализирует:**
- Коммиты за неделю
- Метрики (LOC, test coverage)
- Что сделано хорошо
- Что улучшить

### 6. /document-release
**Что делает:** Обновление документации
**Использование:** `/document-release`
**Обновляет:**
- README
- ARCHITECTURE
- CHANGELOG
- Проверяет консистентность

### 7. /plan-design-review
**Что делает:** Дизайн-аудит (без браузера)
**Использование:** `/plan-design-review`
**Проверяет:**
- UI/UX паттерны
- AI Slop detection
- Design system inference

### 8. /design-consultation
**Что делает:** Создание дизайн-системы
**Использование:** `/design-consultation`
**Создает:**
- Typography
- Color palette
- Spacing scale
- Design preview

## ❌ НЕ РАБОТАЮТ НА WINDOWS (4 skills)

### /browse
**Причина:** Требует компиляции нативного бинарника
**Ошибка:** `failed to copy bun executable into temporary file`

### /qa
**Причина:** Использует /browse для тестирования
**Альтернатива:** Используйте `/taxi-test` или ручное тестирование

### /qa-only
**Причина:** Использует /browse для тестирования
**Альтернатива:** Ручное тестирование по чеклисту

### /setup-browser-cookies
**Причина:** Требует browse binary
**Альтернатива:** Не требуется для других skills
