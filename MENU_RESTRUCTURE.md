## Логика доступа

- Кнопка "🚦 Пробки" в "Мой кабинет" показывает 🔒 для Free пользователей
- Проверка доступа осуществляется при клике на кнопку

## Деплой

```bash
cd /opt/taxibot && git pull origin main && systemctl restart taxibot
```

## Тестирование

После деплоя проверить:
1. `/start` - главное меню показывает 7 кнопок
2. "👤 Мой кабинет" - открывается подменю с 5 пунктами
3. "⚙️ Настройки" - открывается подменю с 2 пунктами
4. Навигация "◀️ Назад" работает корректно
5. Для Free пользователей "🚦 Пробки" показывает замок
6. Для Pro+ пользователей "🚦 Пробки" работает без замка
