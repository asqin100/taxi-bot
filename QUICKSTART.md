# 🚕 Taxi Bot - Быстрый старт

## Запуск бота (3 способа)

### 1. Полный автозапуск (РЕКОМЕНДУЕТСЯ)
```bash
start_all.bat
```
Запускает: веб-сервер → туннель → бот

### 2. Раздельный запуск
```bash
# Терминал 1: Веб-сервер
python -m bot.web.server

# Терминал 2: Туннель (выбрать один)
cloudflared tunnel --url http://localhost:8080  # Без предупреждений
ngrok http 8080                                  # С предупреждением

# Терминал 3: Бот
python -m bot.main
```

### 3. Только бот (без карты)
```bash
python -m bot.main
```

## Решение проблемы ngrok предупреждения

### ❌ Проблема
При открытии карты пользователи видят:
```
You are about to visit...
[Visit Site] button
```

### ✅ Решение: Cloudflare Tunnel

**Шаг 1:** Скачать cloudflared
- https://github.com/cloudflare/cloudflared/releases/latest
- Скачать `cloudflared-windows-amd64.exe`
- Переименовать в `cloudflared.exe`
- Положить в `C:\taxi-bot\`

**Шаг 2:** Запустить
```bash
start_all.bat
# Выбрать опцию 1 (Cloudflare)
```

**Шаг 3:** Скопировать URL
```
Your quick Tunnel has been created! Visit it at:
https://random-name-1234.trycloudflare.com
```

**Шаг 4:** Обновить WebApp в боте
```
/setwebapp https://random-name-1234.trycloudflare.com
```

**Готово!** Теперь карта открывается без предупреждений.

## Основные команды бота

### Пользовательские
- `/start` - Начало работы
- `/menu` - Главное меню
- `/map` - Открыть карту заказов
- `/subscription` - Мой тариф
- `/referral` - Реферальная программа
- `/export` - Экспорт смен (Elite)
- `/heatmap` - Тепловая карта (Elite)

### Админские
- `/admin` - Админ-панель
- Управление промокодами
- Управление балансами
- Управление подписками

## Промокоды

### Типы промокодов
1. **Активация** - дает бесплатную подписку на N дней
2. **Скидка** - дает скидку в % или ₽ на выбранные тарифы

### Создание через админ-панель
1. Открыть `/admin`
2. Раздел "Промокоды"
3. Заполнить форму
4. Создать

## Масштабирование

### Текущая конфигурация
✅ Поддерживает до 100 активных пользователей

### Для 500+ пользователей
- Перейти на PostgreSQL
- Использовать webhook вместо polling
- Настроить rate limiting для Yandex API
- Развернуть на VPS
