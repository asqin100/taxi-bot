# БЫСТРАЯ ИНСТРУКЦИЯ: Определить правильные URL для Robokassa

## Шаг 1: Подключитесь к серверу и обновите код

```bash
ssh root@5.42.110.16
cd /opt/taxibot
git pull origin main
```

## Шаг 2: Запустите диагностику

```bash
bash find_correct_urls.sh
```

Скрипт покажет, какие URL нужно использовать.

## Шаг 3: Скопируйте правильные URL

Скрипт выдаст один из двух вариантов:

### Вариант А: БЕЗ порта (если nginx настроен)
```
Result URL:  http://5.42.110.16/webhook/robokassa/result
Success URL: http://5.42.110.16/webhook/robokassa/success
Fail URL:    http://5.42.110.16/webhook/robokassa/fail
```

### Вариант Б: С портом :8080 (если nginx НЕ настроен)
```
Result URL:  http://5.42.110.16:8080/webhook/robokassa/result
Success URL: http://5.42.110.16:8080/webhook/robokassa/success
Fail URL:    http://5.42.110.16:8080/webhook/robokassa/fail
```

## Шаг 4: Обновите настройки в Robokassa

1. Зайдите в личный кабинет Robokassa
2. Перейдите в настройки магазина
3. Вставьте URL, которые показал скрипт
4. **ОБЯЗАТЕЛЬНО нажмите "Сохранить"!**

## Шаг 5: Протестируйте оплату

1. Откройте бота в Telegram
2. Выберите подписку
3. Оплатите тестовой картой: **5555 5555 5555 5599**
4. После оплаты должно произойти:
   - ✅ Открывается страница "Оплата успешна"
   - ✅ Бот присылает сообщение "✅ Подписка активирована!"

---

**ВАЖНО:** Если скрипт показывает ошибку, сначала проверьте:
```bash
sudo systemctl status taxibot
netstat -tlnp | grep 8080
```
