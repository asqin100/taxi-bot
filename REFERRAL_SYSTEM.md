# Реферальная система - Документация

## Обзор

2-уровневая реферальная система с возможностью оплаты подписки с реферального баланса.

## Основные возможности

### 1. Реферальные ссылки
- Каждый пользователь получает уникальный реферальный код
- Ссылка формата: `https://t.me/BOT_USERNAME?start=REFERRAL_CODE`
- Код генерируется автоматически при первом обращении к реферальной программе

### 2. Комиссионная структура

**Уровень 1 (прямые рефералы):**
- 30% от стоимости подписки
- Начисляется при каждой оплате реферала

**Уровень 2 (рефералы рефералов):**
- 10% от стоимости подписки
- Начисляется при оплате рефералов ваших рефералов

### 3. Бонусы

**Бонус за первую оплату:**
- +100₽ при первой оплате любого реферала

**Бонусы за количество рефералов:**
- 5 рефералов → +200₽
- 10 рефералов → +500₽
- 25 рефералов → +1500₽
- 50 рефералов → +5000₽

### 4. Использование баланса

Реферальный баланс можно использовать для:
- Оплаты подписки Pro (299₽/мес)
- Оплаты подписки Premium (499₽/мес)
- Оплаты подписки Elite (999₽/мес)

**Важно:** При оплате подписки с реферального баланса реферальные бонусы НЕ начисляются (только при оплате картой).

## Технические детали

### Модели базы данных

**User (обновлена):**
```python
referral_code: str | None  # Уникальный реферальный код
referrer_id: int | None    # ID реферера
referral_balance: float    # Баланс в рублях
```

**ReferralEarning:**
```python
user_id: int              # Кому начислено
amount: float             # Сумма в рублях
earning_type: EarningType # Тип начисления
from_user_id: int | None  # От кого получено
subscription_tier: str    # Тариф подписки
```

### API

**Основные функции (bot/services/referral.py):**

- `get_or_create_referral_code(user_id)` - Получить/создать реферальный код
- `register_referral(user_id, referral_code)` - Зарегистрировать реферала
- `process_subscription_payment(user_id, tier, amount)` - Обработать оплату и начислить бонусы
- `withdraw_from_balance(user_id, amount)` - Списать с баланса
- `get_referral_stats(user_id)` - Получить статистику
- `get_balance(user_id)` - Получить баланс

### Хендлеры (bot/handlers/referral.py)

- `/referral` - Показать реферальную программу
- `menu:referral` - Callback для меню
- `referral:pay_subscription` - Оплата подписки с баланса
- `referral:buy_balance:pro` - Купить Pro с баланса
- `referral:buy_balance:premium` - Купить Premium с баланса
- `referral:buy_balance:elite` - Купить Elite с баланса

### Интеграция

**1. Start handler (bot/handlers/start.py):**
- Обрабатывает deep links с реферальными кодами
- Регистрирует новых пользователей как рефералов

**2. Payment service (bot/services/payment.py):**
- После успешной оплаты через YooKassa вызывает `process_subscription_payment`
- Начисляет бонусы рефереру уровня 1 и уровня 2

**3. Main menu (bot/keyboards/inline.py):**
- Добавлена кнопка "🎁 Реферальная программа"

## Миграция базы данных

Для добавления реферальных полей в существующую базу:

```bash

```

Миграция добавляет:
- Колонки: `referral_code`, `referrer_id`, `referral_balance`
- Индексы для оптимизации запросов

## Безопасность

1. **Защита от самореферальства:** Пользователь не может использовать свой собственный реферальный код
2. **Уникальность кодов:** Реферальные коды уникальны (проверка при генерации)
3. **Защита от циклов:** Оплата с реферального баланса не начисляет бонусы
4. **Однократная регистрация:** Пользователь может быть зарегистрирован только один раз как реферал

## Конфигурация

