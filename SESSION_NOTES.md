# Сессия 2026-03-04

## Решена проблема: Ngrok предупреждение

### Проблема
Каждый новый пользователь при открытии карты видел предупреждение ngrok:
```
You are about to visit...
[Visit Site] button
```
Требовалось нажать кнопку для доступа к карте.

### Решение: Cloudflare Tunnel

**Созданные файлы:**
1. `start_cloudflare_tunnel.bat` - скрипт запуска Cloudflare туннеля
2. `start_all.bat` - полный автозапуск (веб-сервер + туннель + бот)
3. `TUNNEL_SETUP.md` - подробная инструкция по настройке
4. `QUICKSTART.md` - быстрый старт для всего проекта

**Обновленные файлы:**
- `DEPLOYMENT_NOTES.md` - добавлен раздел про туннель

### Преимущества Cloudflare Tunnel
- ✅ Полностью бесплатно
- ✅ Без предупреждений
- ✅ Стабильнее ngrok
- ✅ Подходит для 100+ пользователей

### Инструкция для пользователя
1. Скачать cloudflared.exe с GitHub
2. Положить в папку проекта
3. Запустить `start_all.bat`
4. Выбрать опцию 1 (Cloudflare)
5. Скопировать URL из консоли
6. Обновить WebApp: `/setwebapp https://url.trycloudflare.com`

### Альтернативы
- Ngrok Paid ($8/мес) - убирает предупреждение
- VPS (200₽/мес) - для продакшена с 100+ юзерами
