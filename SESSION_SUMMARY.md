## 🚀 Готово к деплою

### Яндекс.Пробки

**Команда:**
```bash
chmod +x /opt/taxibot/update_bot.sh && cd /opt/taxibot && ./update_bot.sh
```

**Что произойдет:**
1. Скачает код с GitHub
2. Остановит бота
3. Запустит бота с Яндекс.Пробками
4. Покажет статус

**Проверка после деплоя:**
```bash
tail -f /opt/taxibot/bot.log | grep traffic
```

Ожидаемый вывод:
```
INFO:bot.services.traffic:Got traffic from Yandex tiles (avg of 4 points): level=X
```
