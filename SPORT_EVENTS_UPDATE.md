# Обновление: Добавлены спортивные мероприятия

## Что изменено

### 1. Парсер событий (`bot/services/event_parser.py`)

**Добавлен парсинг спортивных событий:**
- Теперь бот парсит не только концерты, но и спортивные мероприятия из KudaGo API
- Категории: `["concert", "sport"]`

**Улучшено определение типа события:**
- Расширен список ключевых слов для спортивных событий
- Добавлены: футбол, хоккей, баскетбол, волейбол, теннис, бокс, ММА
- Добавлены названия команд: Спартак, ЦСКА, Динамо, Локомотив, Зенит
- Добавлены турниры: РФПЛ, КХЛ, ВТБ, Евролига, Лига Чемпионов

**Обновлена длительность событий:**
- Спортивные события: 2.5 часа (реалистично для футбола/хоккея)

### 2. Стадионы уже добавлены в venues.json

В базе уже есть все крупные спортивные площадки:
- **Лужники** (south) - главный стадион
- **ВТБ Арена** (north) - Динамо
- **Спартак Стадион** (northwest) - Открытие Арена
- **Локомотив Стадион** (northeast) - РЖД Арена
- **ЦСКА Арена** (northwest)
- **Стадион Торпедо** (east)
- **Стадион Химки** (northwest)
- **Ледовый Дворец ВТБ** (north) - хоккей

### 3. Уведомления

Система уведомлений уже настроена:
- ⚽ Эмодзи для спортивных событий
- Уведомления за 20 минут до окончания (только для Pro/Premium/Elite)
- Уведомления после окончания (для всех пользователей)
- Пользователи могут включить/выключить уведомления о спорте в настройках

## Как это работает

1. **Автоматический парсинг** - каждый час бот проверяет KudaGo API на новые спортивные события
2. **Определение зоны** - по названию стадиона определяется зона Москвы
3. **Уведомления** - пользователи получают алерты о высоком спросе на такси:
   - За 20 минут до окончания матча (платные подписки)
   - Сразу после окончания с текущим коэффициентом (все пользователи)

## Примеры событий, которые будут парситься

- Футбол: "Спартак - ЦСКА", "Локомотив - Зенит"
- Хоккей: "ЦСКА - Динамо", "Спартак - Ак Барс"
- Баскетбол: "ЦСКА - Зенит (ВТБ)"
- Другие: теннис, бокс, волейбол

## Тестирование

После деплоя на сервер:
1. Проверьте логи: `tail -f bot.log | grep -i sport`
2. Запустите парсинг вручную (если есть админ-команда)
3. Проверьте команду `/events` - должны появиться спортивные мероприятия с эмодзи ⚽
