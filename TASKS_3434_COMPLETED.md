## ✅ Task #4: Исправить лимит "Куда ехать"

**Статус:** Выполнено

**Проблема:**
- FREE пользователи могли использовать функцию >3 раз/день
- Счётчик инкрементировался ПОСЛЕ поиска зоны
- Если зона не найдена, счётчик не увеличивался

**Решение:**
- Перенесён вызов `record_where_to_go_use()` сразу после проверки лимита
- Теперь каждая попытка использования считается, даже если зона не найдена
- Счётчик показывается пользователю: "Использовано сегодня: X/Y"

**Файлы:**
- `bot/handlers/location.py` - исправлена логика записи использования

**Коммит:** b3c442d "Исправлен лимит 'Куда ехать': счётчик теперь инкрементируется до поиска"

**Лимиты по тарифам:**
- FREE: 3 запроса/день
- PRO: 7 запросов/день
- PREMIUM: 20 запросов/день
- ELITE: безлимит

## 🚀 Деплой

Все изменения запушены в GitHub (ветка main).

**Команда для деплоя на сервер:**

```bash
cd /opt/taxibot && git pull origin main && systemctl restart taxibot && systemctl status taxibot
```

**Проверка после деплоя:**

1. **Сортировка коэффициентов:**
   - Дождитесь уведомления о высоких коэффициентах
   - Нажмите кнопку "🔤 Сортировать по имени" или "📊 Сортировать по коэффициенту"
   - Проверьте, что список пересортировался

2. **Поиск ресторанов:**
   - Главное меню → "🍔 Заехать на обед"
   - Отправьте геолокацию
   - Должен показать рестораны в радиусе 10 км

3. **Лимит "Куда ехать":**
   - Главное меню → "🔍 Куда ехать"
   - Используйте 3 раза (для FREE)
   - На 4-й раз должно показать "Лимит исчерпан"

4. **Счётчик геоалертов:**
   - Включите геоалерты
   - Дождитесь уведомления
   - Проверьте наличие строки "📊 Использовано сегодня: X/Y"

5. **Бан-система:**
   - Откройте https://kefpulse.ru/admin/dashboard
   - Вкладка "🚫 Забаненные"
   - Попробуйте забанить тестового пользователя

6. **Кнопка VPN:**
   - Главное меню
   - Проверьте, что кнопки "🔐 VPN" нет
