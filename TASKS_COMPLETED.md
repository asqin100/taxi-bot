## Особенности реализации

### Фильтры коэффициентов:
- Текстовый поиск по имени зоны (регистронезависимый)
- Фильтр по минимальному коэффициенту
- Комбинированная фильтрация
- Кнопка сброса фильтров

### Функция обеда:
- Поиск через Yandex Geocoder API
- Радиус: 5 км
- Рестораны: Вкусно и точка, Бургер Кинг, Ростикс
- Навигация: Яндекс.Карты и Яндекс.Навигатор

### Лимиты "Куда ехать":
- FREE: 3 раза/день
- PRO: 7 раз/день
- PREMIUM: 20 раз/день
- ELITE: без ограничений
- Автосброс в полночь
- Показ оставшихся использований

### Бан-система:
- Middleware проверяет бан перед обработкой
- Админы не подвержены бану
- UI в админ-панели
- Команды: /ban, /unban, /banlist

### Минимальная цена:
- Извлекается из Yandex Go API
- Формат: "Коэф: 2.5x | Мин. цена: 350₽"
- Обновляется вместе с коэффициентами

### Business за МКАД:
- Радиус МКАД: 19 км от центра Москвы
- Фильтруется в высоких коэффициентах
- Остаётся доступен внутри МКАД
