# План тестирования KefPulse Bot

## Подготовка к тестированию

### 1. Установка зависимостей

```bash
cd C:\Users\Пользо\taxi-bot
pip install -r requirements.txt
```

Если нет виртуального окружения, создайте:
```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

### 2. Запуск миграций

```bash
python migrate_event_filters.py
python migrate_quiet_hours.py
python migrate_financial.py
```

Ожидаемый результат: "✅ Migration complete!"

## Известные ограничения

1. **Поиск по адресу** - требует YANDEX_GEOCODER_KEY
2. **Пробки** - используют заглушки (нужен реальный API)
3. **KudaGo API** - может не найти события для всех площадок

## Что делать при ошибках

### Ошибка: ModuleNotFoundError
```bash
pip install -r requirements.txt
```

### Ошибка: Database locked
```bash
# Остановить бота
# Удалить data/bot.db
# Запустить миграции заново
```

### Ошибка: API key not configured
```bash
# Добавить в .env:
YANDEX_GEOCODER_KEY=your_key
```
