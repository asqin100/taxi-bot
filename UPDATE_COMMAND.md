# Команда для обновления бота на сервере

## Первый раз (установка скрипта):
```bash
cd /opt/taxibot && git pull origin main && chmod +x update_bot.sh && ./update_bot.sh
```

## Все последующие обновления (одна команда):
```bash
cd /opt/taxibot && ./update_bot.sh
```

Скрипт автоматически:
- Скачивает последний код с GitHub
- Останавливает бота
- Выполняет миграции базы данных
- Запускает бота
- Показывает статус и логи
