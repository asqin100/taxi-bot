# Вариант А: Поиск + Расписание алертов

## ✅ Реализовано

### 1. 🔍 Поиск по адресу/метро

**Файлы:**
- `bot/services/geocoder.py` - геокодинг через Яндекс.Геокодер API
- `bot/handlers/search.py` - обработчик команды `/search`
- `bot/config.py` - добавлен `yandex_geocoder_key`

**Функционал:**
- Команда `/search <адрес>` - поиск коэффициента по адресу
- Примеры: `/search Красная площадь`, `/search м. Тверская`, `/search Шереметьево`
- Геокодинг адреса в координаты
- Определение ближайшей зоны
- Показ текущих коэффициентов для этой зоны
- Кнопки: "Установить алерт", "Показать на карте" (в разработке)

**Как работает:**
1. Пользователь вводит `/search Лужники`
2. Бот геокодирует адрес через Яндекс.Геокодер
3. Находит ближайшую зону по координатам (Haversine formula)
4. Показывает коэффициенты для этой зоны

### 2. ⏰ Расписание алертов (тихие часы)

**Файлы:**
- `bot/models/user.py` - добавлены поля: `quiet_hours_enabled`, `quiet_hours_start`, `quiet_hours_end`
- `bot/services/notification_utils.py` - утилита `is_quiet_hours()`
- `bot/services/notifier.py` - обновлен для учета тихих часов
- `bot/services/event_notifier.py` - обновлен для учета тихих часов
- `bot/handlers/notifications.py` - UI для настройки тихих часов
- `bot/keyboards/inline.py` - клавиатура для выбора времени
- `migrate_quiet_hours.py` - миграция БД

**Функционал:**
- Настройка тихих часов в `/notify`
- Выбор времени начала и окончания (00:00 - 23:00)
- Автоматическая проверка перед отправкой уведомлений
- Поддержка периодов через полночь (например, 22:00 - 07:00)
- Отдельная статистика: сколько уведомлений пропущено из-за тихих часов

**Как работает:**
1. Пользователь открывает `/notify`
2. Включает "Тихие часы"
3. Выбирает время начала (например, 22:00)
4. Выбирает время окончания (например, 07:00)
5. Бот не будет отправлять уведомления с 22:00 до 07:00

## 🚀 Запуск и тестирование

### Шаг 1: Миграция базы данных

```bash
# Миграция для фильтров событий (если еще не запускали)
python migrate_event_filters.py

# Миграция для тихих часов
python migrate_quiet_hours.py
```

### Шаг 2: Настройка .env

Добавьте в `.env`:
```
YANDEX_GEOCODER_KEY=your_geocoder_api_key_here
```

**Получение ключа:**
1. Перейдите на https://developer.tech.yandex.ru/
2. Зарегистрируйтесь и создайте приложение
3. Подключите API "Геокодер"
4. Скопируйте API ключ

### Шаг 3: Запуск бота

```bash
python bot/main.py
```

### Шаг 4: Тестирование

**Тест поиска:**
```
/search Красная площадь
/search м. Тверская
/search Шереметьево
/search Лужники
```

**Тест тихих часов:**
1. `/notify`
2. Включить "Тихие часы"
3. Настроить время (например, 22:00 - 07:00)
4. Проверить, что уведомления не приходят в это время
