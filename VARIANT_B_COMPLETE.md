## 📋 Команды

### Управление сменами

**`/shift_start`** - начать смену
- Запускает таймер смены
- Проверяет, нет ли уже активной смены

**`/shift_end`** - завершить смену
- Интерактивный процесс ввода данных:
  1. Заработок (руб)
  2. Пробег (км)
  3. Количество поездок
  4. Заметки (опционально)
- Автоматический расчет:
  - Расход топлива (на основе пробега)
  - Комиссия агрегатора (% от заработка)
  - Аренда авто (полная или половина, в зависимости от длительности)
  - Чистый доход (заработок - все расходы)
  - Почасовая ставка

**`/stats`** - статистика
- Показывает данные за:
  - Сегодня
  - Неделю
  - Месяц
- Для каждого периода:
  - Количество смен
  - Общее время работы
  - Чистый доход
  - Средняя почасовая ставка
  - Прогресс по целям (если установлены)

---

### Настройка расходов

**`/expenses`** - показать текущие настройки

**`/set_fuel <цена>`** - цена топлива (руб/л)
- Пример: `/set_fuel 55.5`

**`/set_consumption <расход>`** - расход топлива (л/100км)
- Пример: `/set_consumption 8.5`

**`/set_rent <сумма>`** - аренда авто (руб/день)
- Пример: `/set_rent 1500`
- Или `/set_rent 0` если свой автомобиль

**`/set_commission <процент>`** - комиссия агрегатора (%)
- Пример: `/set_commission 20`

---

### Финансовые цели

**`/goal`** - показать текущие цели

**`/set_daily_goal <сумма>`** - дневная цель
- Пример: `/set_daily_goal 5000`

**`/set_weekly_goal <сумма>`** - недельная цель
- Пример: `/set_weekly_goal 30000`

**`/set_monthly_goal <сумма>`** - месячная цель
- Пример: `/set_monthly_goal 120000`

## 💡 Автоматические расчеты

### Расход топлива
```
Формула: (пробег_км / 100) × расход_на_100км × цена_за_литр

Пример:
Пробег: 180 км
Расход: 9 л/100км
Цена: 56 руб/л

Топливо = (180 / 100) × 9 × 56 = 907.20 руб
```

### Комиссия
```
Формула: заработок × (комиссия_процент / 100)

Пример:
Заработок: 6500 руб
Комиссия: 18%

Комиссия = 6500 × 0.18 = 1170 руб
```

### Аренда
```
Если смена >= 12 часов: полная аренда
Если смена < 12 часов: половина аренды

Пример:
Аренда: 1500 руб/день
Смена: 10.5 ч → 750 руб
Смена: 12.5 ч → 1500 руб
```

### Чистый доход
```
Формула: заработок - топливо - комиссия - аренда - прочее

Пример:
Заработок: 6500 руб
Топливо: 907.20 руб
Комиссия: 1170 руб
Аренда: 1500 руб

Чистый = 6500 - 907.20 - 1170 - 1500 = 2922.80 руб
```

## 📊 База данных

### Таблица `shifts`
- `id` - ID смены
- `user_id` - Telegram ID пользователя
- `start_time` - Время начала
- `end_time` - Время окончания
- `gross_earnings` - Заработок до вычетов
- `trips_count` - Количество поездок
- `distance_km` - Пробег
- `fuel_cost` - Расход на топливо
- `rent_cost` - Аренда
- `commission` - Комиссия
- `other_expenses` - Прочие расходы
- `net_earnings` - Чистый доход
- `notes` - Заметки

### Таблица `user_financial_settings`
- `id` - ID настройки
- `user_id` - Telegram ID пользователя
- `fuel_price_per_liter` - Цена топлива
- `fuel_consumption_per_100km` - Расход
- `rent_per_day` - Аренда
- `commission_percent` - Комиссия
- `daily_goal` - Дневная цель
- `weekly_goal` - Недельная цель
- `monthly_goal` - Месячная цель
