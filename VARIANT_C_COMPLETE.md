# Вариант В: Дорожная обстановка

## ✅ Реализовано

### 🚦 Дорожная обстановка - интеграция с пробками

**Файлы:**
- `bot/services/traffic.py` - сервис для работы с данными о пробках
- `bot/handlers/traffic.py` - обработчики команд

---

## 📋 Команды

**`/traffic`** - общая дорожная обстановка
- Показывает текущие пробки в Москве
- Данные по МКАД и ТТК
- Рекомендации на основе коэффициентов и пробок
- Обновляется каждые 5 минут (кэш)

**`/traffic_mkad`** - пробки на МКАД
- Детальная информация о загруженности МКАД
- Рекомендации по объездным путям
- Уровень пробок 1-10 баллов

**`/traffic_ttk`** - пробки на ТТК
- Детальная информация о загруженности ТТК
- Рекомендации по альтернативным маршрутам
- Уровень пробок 1-10 баллов

---

## 🎨 Визуализация

### Уровни пробок

- 🟢 **1-3 балла** - Свободно
- 🟡 **4-6 баллов** - Средние пробки
- 🟠 **7-8 баллов** - Затруднено
- 🔴 **9-10 баллов** - Серьезные пробки

### Умные рекомендации

Бот анализирует комбинацию пробок и коэффициентов:

- ✅ **Высокий кэф + нормальные дороги** - "Отличное время!"
- ⚠️ **Высокий кэф + пробки** - "Будьте готовы к задержкам"
- ❌ **Низкий кэф + пробки** - "Не рекомендуется"
- 🟢 **Свободные дороги** - "Хорошее время для работы"

## 🔄 Кэширование

Данные о пробках кэшируются на **5 минут**, чтобы:
- Снизить нагрузку на API
- Ускорить ответы бота
- Избежать лимитов запросов

При повторном запросе в течение 5 минут бот вернет кэшированные данные.

---

## 🔗 Интеграция с другими функциями

### С коэффициентами

Команда `/traffic` автоматически учитывает текущие коэффициенты и дает рекомендации:

```python
if coefficient >= 2.0 and traffic_level <= 5:
    return "✅ Отличное время! Высокий кэф и нормальные дороги"
elif coefficient >= 2.0 and traffic_level > 5:
    return "⚠️ Высокий кэф, но пробки. Будьте готовы к задержкам"
```

### С поиском

После `/search <адрес>` можно проверить пробки в этом районе.

### С финансовым трекером

Пробки влияют на:
- Расход топлива (больше простоя = больше расход)
- Количество поездок (меньше поездок в час)
- Почасовую ставку

## ⚙️ Настройка API

### Яндекс.Пробки API

Для работы с реальными данными нужен API ключ:

1. Перейдите на https://developer.tech.yandex.ru/
2. Создайте приложение
3. Подключите API "Пробки" или "Карты"
4. Добавьте ключ в `.env`:

```
YANDEX_API_KEY=your_traffic_api_key_here
```

### Альтернативные источники

Если Яндекс API недоступен, можно использовать:
- Google Maps Traffic API
- HERE Traffic API
- TomTom Traffic API
- OpenStreetMap данные
