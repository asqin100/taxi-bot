# Анализ: Парсинг пробок с Яндекс.Карт

## Техническая возможность

✅ **Технически возможно** - создал proof-of-concept в `yandex_traffic_parser_poc.py`

### Как это работает:

1. **Получение тайлов:**
   - Endpoint: `https://core-renderer-tiles.maps.yandex.net/tiles?l=trf`
   - Конвертируем lat/lon в координаты тайла (x, y, zoom)
   - Скачиваем PNG изображение слоя пробок

2. **Анализ цветов:**
   - Зеленый → свободно (1-3 балла)
   - Желтый → средние пробки (4-6 баллов)
   - Оранжевый → затруднено (7-8 баллов)
   - Красный → серьезные пробки (9-10 баллов)

3. **Вычисление уровня:**
   - Подсчитываем пиксели каждого цвета
   - Вычисляем взвешенный средний уровень

## ⚠️ КРИТИЧЕСКИЕ ПРОБЛЕМЫ

### 1. Юридические риски

❌ **Нарушение Terms of Service Яндекса**
- Парсинг запрещен пользовательским соглашением
- Риск судебных исков
- Яндекс может заблокировать IP/сервер

### 2. Технические проблемы

❌ **Нестабильность:**
- Endpoint может измениться в любой момент
- Формат тайлов может поменяться
- Цвета могут быть другими

❌ **Производительность:**
- Нужно скачивать PNG для каждой точки
- Обработка изображений = CPU нагрузка
- Медленнее, чем API запрос

❌ **Точность:**
- Цвета могут варьироваться
- Артефакты сжатия PNG
- Зависит от zoom level
- Менее точно, чем официальный API

### 3. Операционные риски

❌ **Блокировка:**
- Яндекс может заблокировать IP при частых запросах
- Нужны rate limits и задержки
- Может потребоваться ротация IP

❌ **Поддержка:**
- Сложный код для поддержки
- Нужна библиотека Pillow
- Больше точек отказа

## Рекомендация

### ❌ НЕ РЕКОМЕНДУЮ использовать парсинг Яндекса

**Причины:**
1. Юридические риски (нарушение ToS)
2. Технические проблемы (нестабильность, сложность)
3. Операционные риски (блокировка IP)
4. TomTom работает отлично и использует только 29% лимита

### ✅ РЕКОМЕНДУЮ оставить TomTom

**Текущее решение:**
- Легально и стабильно
- 720 запросов/день (29% лимита)
- Простой и надежный код
- Хорошее покрытие Москвы
- Нет рисков

## Альтернатива: HERE Maps

Если очень хочется сменить провайдера:

**HERE Maps Traffic API:**
- ✅ Легально
- ✅ 250,000 запросов/месяц бесплатно (~8,300/день)
- ✅ Официальный REST API
- ✅ Хорошее покрытие России
- ⚠️ Нужна регистрация

## Итог

Парсинг Яндекса технически возможен (см. `yandex_traffic_parser_poc.py`), но **категорически не рекомендуется** для production из-за юридических и технических рисков.

**Лучшее решение:** Оставить TomTom или переключиться на HERE Maps (если нужно больше запросов).
