# Инструкция: Подключение Яндекс API для пробок

## Шаг 1: Получить API ключ

В твоем кабинете разработчика (скриншот 2929.png):

1. Нажми "Подключить API"
2. Выбери: **"Маршрутизация: Планирование и Мониторинг"**
3. Нажми "Продолжить"
4. Скопируй полученный API ключ

## Шаг 2: Добавить ключ в .env

Открой файл `.env` на сервере и добавь:

```bash
YANDEX_ROUTER_API_KEY=твой_ключ_здесь
```

## Шаг 3: Тестирование

Запусти тестовый скрипт локально:

```bash
cd C:\taxi-bot
python test_yandex_router.py
```

Если API работает, мы увидим ответ с duration (временем маршрута).

## Шаг 4: Реализация

Если тест успешен, я реализую:
1. Функцию для запроса маршрутов в разных точках Москвы
2. Вычисление уровня пробок из соотношения времени
3. Замену TomTom на Яндекс в traffic.py

## Важно

⚠️ Этот метод - **workaround**, не прямой Traffic API:
- Менее точный, чем TomTom Flow API
- Больше запросов (нужен маршрут для каждой точки)
- Сложнее в реализации

Если не сработает - можем попробовать HERE Maps (250k запросов/месяц бесплатно).
