Откройте бота в Telegram и отправьте команду:

```

```

Это обновит URL карты в боте.

## 🔧 Полезные команды

### Управление ботом
```bash
# Перезапуск
systemctl restart taxibot

# Статус
systemctl status taxibot

# Логи в реальном времени
journalctl -u taxibot -f

# Логи в файле
tail -f /var/log/taxibot/bot.log
```

### Обновление кода
```bash
cd /opt/taxibot
sudo -u taxibot git pull
systemctl restart taxibot
```

### Мониторинг
```bash
# Проверка всех сервисов
bash /opt/taxibot/deploy/monitor.sh

# Проверка веб-сервера
curl http://localhost:8080
```

### Бэкап базы данных
```bash
bash /opt/taxibot/deploy/backup.sh
```
