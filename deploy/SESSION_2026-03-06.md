### 5. Исправленные проблемы

#### Проблема 1: Несоответствие версий Python
- **Ошибка**: venv использовал Python 3.12, пакеты устанавливались для 3.11
- **Решение**: Пересоздан venv с Python 3.11

#### Проблема 2: Отсутствующие зависимости
- **Ошибка**: ModuleNotFoundError для yookassa, matplotlib, seaborn
- **Решение**: Добавлены в requirements.txt и установлены

#### Проблема 3: config.py не поддерживал production поля
- **Ошибка**: ValidationError для admin_ids, database_url, web_host, redis_url, environment
- **Решение**: Обновлен config.py с добавлением всех полей и `extra="allow"`

#### Проблема 4: db.py использовал неправильное поле
- **Ошибка**: Пустой db_url вместо database_url
- **Решение**: Изменено на `settings.effective_db_url`

#### Проблема 5: Telegram требует HTTPS для WebApp
- **Ошибка**: `Bad Request: inline keyboard button Web App URL 'http://5.42.110.16' is invalid: Only HTTPS links are allowed`
- **Решение**: Настроен домен kefpulse.ru с SSL сертификатом

#### Проблема 6: Кнопка мини-апп показывает ngrok ошибку
- **Причина**: Menu Button URL в BotFather не обновлен
- **Решение**: Нужно обновить через @BotFather → Bot Settings → Menu Button → Edit Menu Button URL → https://kefpulse.ru

#### Проблема 7: TOP-5 зон показывает все тарифы
- **Ошибка**: Пользователь выбирает в настройках "Бизнес" и "Комфорт", но TOP-5 показывает и "Эконом"
- **Решение**: Обновлен handlers/coefficients.py - теперь запрашивает выбранные тарифы из БД и фильтрует результаты

#### Проблема 8: Поиск доступен всем пользователям
- **Требование**: Функция "Поиск по адресу" должна быть доступна только в Pro подписке
- **Решение**: Добавлены проверки подписки в handlers/search.py, обновлены клавиатуры с иконкой замка

## 🔧 Полезные команды

### Управление ботом
```bash
# Перезапуск
systemctl restart taxibot

# Статус
systemctl status taxibot

# Логи в реальном времени
journalctl -u taxibot -f
tail -f /var/log/taxibot/bot.log

# Логи ошибок
tail -f /var/log/taxibot/bot-error.log
```

### Обновление кода
```bash
cd /opt/taxibot
sudo -u taxibot git pull
systemctl restart taxibot
```

### Проверка конфигурации
```bash
# Проверить .env
cat /opt/taxibot/.env

# Проверить что Python видит настройки
cd /opt/taxibot && sudo -u taxibot /opt/taxibot/venv/bin/python -c "from bot.config import settings; print(f'webapp_url: {settings.webapp_url}')"
```

### Nginx
```bash
# Проверить конфигурацию
nginx -t

# Перезапустить
systemctl restart nginx

# Логи
tail -f /var/log/nginx/taxibot-access.log
tail -f /var/log/nginx/taxibot-error.log
```

### PostgreSQL
```bash
# Подключиться к БД
sudo -u postgres psql taxibot_db

# Проверить количество пользователей
sudo -u postgres psql taxibot_db -c "SELECT COUNT(*) FROM users;"

# Бэкап
bash /opt/taxibot/deploy/backup.sh
```

### SSL сертификат
```bash
# Проверить срок действия
sudo certbot certificates

# Тест обновления
sudo certbot renew --dry-run

# Принудительное обновление
sudo certbot renew
```
