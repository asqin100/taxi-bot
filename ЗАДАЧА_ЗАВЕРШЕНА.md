### 1. Исправлен timezone баг
- **Проблема:** Мероприятия показывались на 3 часа раньше
- **Причина:** KudaGo API возвращает UTC, код обрабатывал как локальное время
- **Решение:** Создан `bot/utils/timezone.py` с функциями `now()` и `from_timestamp()`
- **Коммит:** 31e8a3f
- **Статус:** ✅ Исправлено и запушено

### 2. Исправлен datetime import баг
- **Проблема:** `NameError: name 'datetime' is not defined` в events.py
- **Причина:** Использовался в type hint, но не был импортирован
- **Решение:** Добавлен `from datetime import datetime, timedelta`
- **Коммит:** 056fbca
- **Статус:** ✅ Исправлено и запушено

### 3. Созданы инструменты для исправления дубликатов

- **Решение:** Созданы скрипты `fix_duplicate_bots.sh` и `RUN_THIS_ON_SERVER.sh`
- **Коммиты:** 8536616, 3c2ca0c
